Voiceflow's Production offering is centred around user interactions. 

An interaction is defined as any user input (e.g. an utterance or button click) that is sent to the Dialog Manager for action. 

## Pay as you go

| Interactions per month | Per Interaction |
| :--------------------- | :-------------- |
| First 5,000            | Free            |
| Next 250,000           | $0.02           |
| Next 750,000           | $0.015          |
| Next 4 Million         | $0.01           |
| Next 10 Million        | $0.0075         |
| Above 15 Million       | $0.0065         |

*Committed usage discounts are available on Enterprise plan*
