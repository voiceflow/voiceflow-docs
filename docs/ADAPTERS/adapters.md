## Overview

Adapters adapt Voiceflow's JSON project structure (.VF) into formats that can be Imported from, or Exported to, 3rd Party Platforms, such as Dialogflow.

- **Import Adapters: **Adapt 3rd Party Platform data into a format that Voiceflow can Import
- **Export Adapters:** Adapt Voiceflow's JSON file into a format that can be Exported into 3rd Party Platforms
