Voiceflow supports native integrations with a wide range of tools—enabling your agent to take real-time action using external services like CRMs, support systems, databases, and automation platforms. These integrations are configured directly within the Agent Step and are triggered dynamically during a conversation.

This makes your assistant not only conversational but operational—capable of resolving support tickets, sending emails, retrieving user data, and much more.

## Available Tools & Use Cases

Below are the currently supported tools:

### Zendesk

* Add ticket comment
* Create ticket
* Find group
* Find latest comment
* Find ticket(s)
* Find user
* Update organization
* Update ticket

### Salesforce

* Add comment to a case
* Create case
* Create contact
* Create lead
* Update case
* Update contact
* Update Salesforce lead

### Google Sheets

* Add to sheet
* Create new spreadsheet
* Get rows
* Get sheet
* Update sheet

### Gmail

* Send email

### Airtable

* Use as a flexible backend to store structured data
* Query tables for user preferences or records
* Update fields with conversation results

### Make (formerly Integromat)

* Trigger workflows for tasks like onboarding, notifications, or file handling
* Connect to hundreds of other apps
* Ideal for complex automation chains

### Twilio

* Send or receive SMS
* Trigger outbound calls
* Use programmable messaging flows

### Hubspot

* Manage leads and contacts
* Log conversations or update properties
* Route users based on CRM stage
