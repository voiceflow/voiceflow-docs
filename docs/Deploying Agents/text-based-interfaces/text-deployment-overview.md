You can use our Dialog Manager API or Knowledge Base API to build fully-customized experiences for your customers on (mostly) text-based channels or interfaces such as a website, WhatsApp, Discord, Slack, SMS, and to your own app.

For quick-start examples, see:

- [WhatsApp](https://developer.voiceflow.com/v2.0/docs/advanced-whatsapp)
- [Discord](https://developer.voiceflow.com/v2.0/docs/discord)
- [Slack](https://developer.voiceflow.com/v2.0/docs/slack)
- [Telegram](https://developer.voiceflow.com/v2.0/docs/how-to-telegram)
- [Twilio SMS](https://developer.voiceflow.com/v2.0/docs/twilio-sms)
- [Microsoft Teams](https://developer.voiceflow.com/v2.0/docs/microsoft-teams-copy)
- [Your own Custom Channel using our APIs](https://developer.voiceflow.com/v2.0/docs/custom-channel-api)
