Leverage our Dialog Manager API or Knowledge Base API in your voice-based integrations such as an Interactive Voice Response (IVR) system or a multi-modal web or mobile experience.

> 📘 Voiceflow does not have a built-in automatic speech recognition (ASR) or speech-to-text (STT) solution. Use a third-party ASR or STT platform to transcribe your users' input before passing it to our APIs to retrieve the next set of responses.
