## Intro

With a guest appearance, dive deeper into Sephora's customer facing bot in North American and Australia. This includes

* An overview of design principles and best practices for a retail assistant
* Common pitfalls of shopping assistants
* Real build time using Voiceflow to redesign core retail flows
