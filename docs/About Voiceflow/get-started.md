Voiceflow allows you to build powerful and flexible AI agents, but you can start building a basic agent in minutes. This   quick start guide will get you up and running with a basic agent with information its in [Knowledge Base](doc:knowledge-base) that you can build upon.

<br />

## You will need

* A free Voiceflow account. [Sign up here](https://creator.voiceflow.com/signup) .

<br />
